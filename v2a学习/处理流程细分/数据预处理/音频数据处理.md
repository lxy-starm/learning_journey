# 音频数据处理思维导图

```
音频数据处理 (Audio Data Processing)
│
├── 1. 涉及文件总览
│   │
│   ├── preprocess/NAT_mel.py
│   │   └── MelNet 网络定义：波形 → 梅尔频谱图的核心计算模块
│   │
│   ├── preprocess/collect_mel_spect.py
│   │   └── 批量梅尔频谱提取脚本：目录级音频批量处理
│   │
│   ├── preprocess/collect_MelSpec.py
│   │   └── 简单单例提取脚本（测试/调试用）
│   │
│   ├── ldm/data/joinaudiodataset_624.py
│   │   └── 训练数据集加载器：TSV → 训练Batch
│   │
│   └── ldm/data/joinaudiodataset_624_new.py
│       └── 新版训练数据集加载器（改进版）
│
├── 2. 梅尔频谱提取核心 — NAT_mel.py (MelNet)
│   │
│   ├── 2.1 类定义：MelNet(nn.Module)
│   │   │
│   │   ├── 初始化参数（hparams 字典）
│   │   │   ├── audio_sample_rate: 16000 Hz（目标采样率）
│   │   │   ├── audio_num_mel_bins: 80（梅尔滤波器组数）
│   │   │   ├── fft_size: 1024（FFT 窗口大小）
│   │   │   ├── win_size: 1024（分析窗口大小）
│   │   │   ├── hop_size: 256（帧移步长）
│   │   │   ├── fmin: 0 Hz（最低频率）
│   │   │   └── fmax: 8000 Hz（最高频率）
│   │   │
│   │   ├── 预计算矩阵（__init__ 中初始化）
│   │   │   ├── mel_basis: librosa_mel_fn() → [80, fft_size/2+1]
│   │   │   │   └── 梅尔滤波器组，将线性频谱映射到梅尔刻度
│   │   │   └── hann_window: torch.hann_window(win_size)
│   │   │       └── 汉宁窗，减少频谱泄漏
│   │   │
│   │   └── 设备迁移
│   │       └── to(device) 时同步迁移 mel_basis 和 hann_window
│   │
│   ├── 2.2 前向传播 — forward(y, center=False, complex=False)
│   │   │
│   │   ├── 输入
│   │   │   ├── y: 音频波形 [1, wav_len]（支持 numpy/tensor）
│   │   │   ├── center: STFT 是否中心对齐（默认 False）
│   │   │   └── complex: 是否返回复数频谱（默认 False）
│   │   │
│   │   ├── Step 1: 输入预处理
│   │   │   ├── 类型转换：numpy → torch.FloatTensor
│   │   │   ├── 维度调整：1D → unsqueeze(0) → [1, wav_len]
│   │   │   └── 幅值裁剪：y.clamp(min=-1., max=1.)
│   │   │
│   │   ├── Step 2: 反射填充（Padding）
│   │   │   ├── pad 量 = (n_fft - hop_size) / 2 = (1024-256)/2 = 384
│   │   │   ├── 模式：reflect（镜像填充，避免边界伪影）
│   │   │   └── y.unsqueeze(1) → pad → squeeze(1)
│   │   │
│   │   ├── Step 3: 短时傅里叶变换 (STFT)
│   │   │   ├── torch.stft(y, n_fft=1024, hop_length=256, win_length=1024)
│   │   │   ├── 窗函数：hann_window
│   │   │   ├── 输出：[B, n_fft/2+1, T, 2]（复数：实部+虚部）
│   │   │   └── T ≈ wav_len / hop_size
│   │   │
│   │   ├── Step 4: 幅度谱计算（complex=False 时）
│   │   │   ├── sqrt(real² + imag² + 1e-9)
│   │   │   │   └── 1e-9 防止梯度爆炸
│   │   │   └── 输出：[B, n_fft/2+1, T] = [1, 513, T]
│   │   │
│   │   ├── Step 5: 梅尔滤波器组映射
│   │   │   ├── torch.matmul(mel_basis, spec)
│   │   │   ├── mel_basis: [80, 513] × spec: [513, T]
│   │   │   └── 输出：[B, 80, T]
│   │   │
│   │   ├── Step 6: 对数压缩（动态范围压缩）
│   │   │   ├── spectral_normalize_torch(spec)
│   │   │   │   └── torch.log(torch.clamp(spec, min=1e-5))
│   │   │   └── log 压缩使大小幅值在相近量级
│   │   │
│   │   └── 输出
│   │       ├── complex=False: 梅尔频谱 [B, 80, T]
│   │       └── complex=True:  复数频谱 [B, T, n_fft, 2]
│   │
│   └── 2.3 辅助函数
│       │
│       ├── dynamic_range_compression(x, C=1, clip_val=1e-5)
│       │   └── log10(max(x, clip_val) * C)  （numpy 版）
│       │
│       ├── dynamic_range_decompression(x, C=1)
│       │   └── exp(x) / C  （numpy 版）
│       │
│       ├── spectral_normalize_torch(magnitudes)
│       │   └── log(clamp(magnitudes, min=1e-5))  （torch 版）
│       │
│       └── spectral_de_normalize_torch(magnitudes)
│           └── exp(magnitudes)  （torch 版）
│
├── 3. 批量梅尔频谱提取 — collect_mel_spect.py
│   │
│   ├── 3.1 数据集类 — tsv_dataset(Dataset)
│   │   │
│   │   ├── __init__(wav_dir, sr, mode, hop_size, target_mel_length)
│   │   │   ├── self.audio_paths = os.listdir(wav_dir)
│   │   │   ├── self.sr = 16000（目标采样率）
│   │   │   ├── self.mode = 'pad'/'tile'/'none'
│   │   │   └── self.target_mel_length = batch_max_length
│   │   │
│   │   ├── __getitem__(index)
│   │   │   ├── torchaudio.load(audio_path) → [channels, wav_len]
│   │   │   ├── 立体声转单声道：wav.mean(0, keepdim=True)
│   │   │   │   └── [2, wav_len] → [1, wav_len]
│   │   │   ├── 重采样：torchaudio.functional.resample(orig_sr → 16000)
│   │   │   └── pad 模式：pad_wav() 填充到目标长度
│   │   │
│   │   └── pad_wav(wav)
│   │       ├── segment_length = (target_mel_length + 1) * hop_size
│   │       ├── 过长：裁剪 wav[:, :segment_length]
│   │       └── 过短：零填充 torch.zeros + 复制原始波形
│   │
│   ├── 3.2 核心处理函数 — process_audio(rank, args)
│   │   │
│   │   ├── 初始化
│   │   │   ├── 创建 tsv_dataset 实例
│   │   │   ├── 创建 DataLoader（batch_size=1, num_workers=16）
│   │   │   ├── 实例化 MelNet 网络
│   │   │   └── MelNet.to(device)
│   │   │
│   │   ├── 遍历处理
│   │   │   ├── 加载音频波形 → wavs.to(device)
│   │   │   ├── mel_net(wav) 提取频谱 → [1, 80, T]
│   │   │   ├── 长度检查与处理
│   │   │   │   ├── 若 mel_len <= batch_max_length
│   │   │   │   │   ├── mode='tile': np.tile 重复填充
│   │   │   │   │   ├── mode='pad'/'none': 保持不变
│   │   │   │   │   └── 裁剪：spec[:, :batch_max_length]
│   │   │   │   └── 跳过已存在的文件
│   │   │   └── np.save(mel_path, spec)
│   │   │
│   │   └── 输出文件命名
│   │       └── wav_name[:-4] + '_mel.npy'
│   │           └── 如：audio001.wav → audio001_mel.npy
│   │
│   ├── 3.3 辅助函数
│   │   │
│   │   ├── drop_bad_wav(item)
│   │   │   └── 检测损坏/过短的音频文件（< 0.1s）
│   │   │
│   │   ├── drop_bad_wavs(tsv_path)
│   │   │   ├── 多进程并行检测（max_workers=16）
│   │   │   ├── 输出 bad_wavs.json
│   │   │   └── 从 TSV 中删除坏文件行
│   │   │
│   │   └── addmel2tsv(save_dir, tsv_path)
│   │       ├── 扫描所有 *_mel.npy 文件
│   │       ├── 建立 name → mel_path 映射
│   │       └── 将 mel_path 列添加到 TSV 文件
│   │
│   ├── 3.4 命令行参数
│   │   │
│   │   ├── --num_gpus: GPU 数量（默认 1）
│   │   ├── --max_duration: 最大音频时长（默认 30s）
│   │   ├── --wave_path: 音频输入目录
│   │   └── --save_path: 频谱保存目录
│   │
│   └── 3.5 主流程 — __main__
│       │
│       ├── 计算频谱最大长度
│       │   └── batch_max_length = max_duration × 62.5
│       │       └── 62.5 = 16000 / 256（每秒对应的梅尔帧数）
│       │       └── 30s → 1875 帧
│       │
│       ├── 构建参数字典
│       │   ├── 音频参数（采样率、FFT、梅尔维度等）
│       │   ├── mode: 'pad'（默认填充模式）
│       │   ├── save_mel: True
│       │   └── save_resample: False
│       │
│       └── 调用 process_audio(0, args)
│
├── 4. 简单提取脚本 — collect_MelSpec.py
│   │
│   └── 功能：单目录音频提取为 .npy 字典
│       ├── 使用 AudioDataset 加载指定目录
│       ├── 逐个提取梅尔频谱
│       ├── 存入字典 {wav_name: wav_spec}
│       └── np.save() 保存为单个 .npy（allow_pickle=True）
│       └── 用途：快速测试/调试，非正式流程
│
├── 5. 训练数据集加载器 — joinaudiodataset_624.py
│   │
│   ├── 5.1 类层次结构
│   │   │
│   │   ├── JoinManifestSpecs(torch.utils.data.Dataset)  ← 基类
│   │   ├── JoinSpecsTrain(JoinManifestSpecs)             ← 训练集
│   │   ├── JoinSpecsValidation(JoinManifestSpecs)        ← 验证集
│   │   └── JoinSpecsTest(JoinManifestSpecs)              ← 测试集
│   │
│   ├── 5.2 JoinManifestSpecs 基类
│   │   │
│   │   ├── __init__(split, spec_dir_path, mel_num, spec_crop_len, drop, ...)
│   │   │   │
│   │   │   ├── TSV 文件加载
│   │   │   │   ├── 扫描目录：glob(f'{dir_path}/**/*.tsv', recursive=True)
│   │   │   │   ├── 支持多目录逗号分隔
│   │   │   │   ├── pd.read_csv(manifest, sep='\t')
│   │   │   │   └── pd.concat() 合并所有 TSV
│   │   │   │
│   │   │   ├── 数据集划分
│   │   │   │   ├── train: df.iloc[100:]（跳过前100条）
│   │   │   │   ├── valid: df.iloc[:100]（前100条）
│   │   │   │   └── test: 全部数据 + add_name_num() 编号
│   │   │   │
│   │   │   └── 关键参数
│   │   │       ├── spec_crop_len: 频谱裁剪长度（默认 1248）
│   │   │       ├── mel_num: 梅尔维度（80）
│   │   │       └── drop: 条件丢弃概率（Classifier-Free Guidance）
│   │   │
│   │   ├── add_name_num(df) — 测试集名称编号
│   │   │   └── 同一音频可能有多个 caption
│   │   │       └── 添加后缀 _0, _1, _2... 区分
│   │   │
│   │   └── __getitem__(idx) — 核心数据加载
│   │       │
│   │       ├── 加载梅尔频谱
│   │       │   ├── spec = np.load(data['mel_path'])
│   │       │   ├── 异常处理：加载失败 → 零填充
│   │       │   └── 长度不足：np.tile 重复填充
│   │       │
│   │       ├── 裁剪到统一长度
│   │       │   └── spec[:, :spec_crop_len] → [80, 1248]
│   │       │
│   │       ├── 条件特征加载
│   │       │   ├── 正常：item["caption"] = data['caption']
│   │       │   │   └── caption 是 CLIP 特征文件路径
│   │       │   └── 随机丢弃（CFG 训练）
│   │       │       ├── p = np.random.uniform(0, 1)
│   │       │       ├── p > drop → 保留条件
│   │       │       └── p ≤ drop → item["caption"] = ""（无条件）
│   │       │
│   │       └── 输出字典
│   │           ├── item["image"]: [80, 1248] 梅尔频谱
│   │           ├── item["caption"]: str（CLIP 特征路径或空串）
│   │           └── item["f_name"]: str（仅测试集）
│   │
│   └── 5.3 与新版差异 — joinaudiodataset_624_new.py
│       └── 代码结构基本一致
│           └── 主要是代码风格微调，逻辑无本质变化
│
├── 6. 数据流全景图
│   │
│   ┌────────────┐
│   │ *.wav      │  原始音频文件
│   │ 采样率各异 │
│   └─────┬──────┘
│         ↓
│   ┌──────────────────────────────────────────┐
│   │  collect_mel_spect.py                    │
│   │  ┌────────────────────────────────────┐  │
│   │  │ tsv_dataset                        │  │
│   │  │  ├── torchaudio.load()             │  │
│   │  │  ├── 立体声→单声道                 │  │
│   │  │  └── resample → 16000 Hz           │  │
│   │  └──────────────┬─────────────────────┘  │
│   │                 ↓                         │
│   │  ┌────────────────────────────────────┐  │
│   │  │ MelNet.forward()                   │  │
│   │  │  ├── 反射填充 (pad=384)            │  │
│   │  │  ├── STFT (fft=1024, hop=256)      │  │
│   │  │  ├── 幅度谱 sqrt(real²+imag²)     │  │
│   │  │  ├── 梅尔滤波器组 [80, 513]×[513,T]│  │
│   │  │  └── log 压缩                      │  │
│   │  └──────────────┬─────────────────────┘  │
│   │                 ↓                         │
│   │         *_mel.npy [80, T]                │
│   └─────────────────┬────────────────────────┘
│                     ↓
│   ┌──────────────────────────────────────────┐
│   │  TSV 文件组织                            │
│   │  name | caption | mel_path               │
│   │  ─────┼─────────┼─────────────────       │
│   │  v001 | clip/v001.npy | mel/v001_mel.npy │
│   └─────────────────┬────────────────────────┘
│                     ↓
│   ┌──────────────────────────────────────────┐
│   │  joinaudiodataset_624.py                 │
│   │  ├── 读取 TSV                            │
│   │  ├── np.load(mel_path) → [80, 1248]     │
│   │  ├── 加载 caption (CLIP特征路径)         │
│   │  ├── 随机条件丢弃 (CFG)                  │
│   │  └── 输出 {"image", "caption", "f_name"} │
│   └─────────────────┬────────────────────────┘
│                     ↓
│              DataLoader → Diffusion Model
│
└── 7. 关键参数速查表
    │
    ├── 音频参数
    │   ├── 采样率：16000 Hz
    │   ├── 梅尔维度：80
    │   ├── FFT 大小：1024
    │   ├── 窗口大小：1024
    │   ├── 帧移步长：256
    │   ├── 最低频率：0 Hz
    │   ├── 最高频率：8000 Hz
    │   └── 每秒梅尔帧数：16000/256 ≈ 62.5
    │
    ├── 频谱尺寸
    │   ├── 10s 音频 → [80, 624]
    │   ├── 训练裁剪长度 → [80, 1248]（约 20s）
    │   ├── VAE 压缩后 → [4, 10, 78]
    │   └── 填充模式：pad / tile / none
    │
    ├── 数据集划分
    │   ├── train: df.iloc[100:]
    │   ├── valid: df.iloc[:100]
    │   └── test: 全部 + 名称编号
    │
    └── 条件丢弃 (CFG)
        ├── drop 概率：0（默认不丢弃）
        ├── p > drop → 保留条件 caption
        └── p ≤ drop → caption = ""（无条件）
```
